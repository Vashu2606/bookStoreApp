# 📚 Book Store App (MERN Stack)

A full-stack Book Store web application built using the MERN stack (MongoDB, Express.js, React.js, Node.js).  
This platform allows users to explore free and paid books with a clean and responsive UI.

---

## 🚀 Features

- 📖 Browse books (Free & Paid)
- 🔐 User Authentication (Login/Signup)
- 🌙 Dark / Light Mode Toggle
- 📱 Fully Responsive Design
- ⚡ Dynamic Data from MongoDB
- 🎯 Clean UI using Tailwind CSS
